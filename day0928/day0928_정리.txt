[변수와 자료형]
@변수란?
실행중에 데이터를 저장하기 위해서는 저장하기 위한 메모리를 확보해야함.
이 메모리를 확보하는 것을 "변수를 선언한다" 라고 한다.

@자료형이란?
변수를 선언할때는 변수에 어떤 성격의 자료를 저장할것인지 정해줘야하며
그것을 '자료형'이라고 함.
   
   *자료형 종류
   기본자료형 : 값을 가지고 있는 변수
   참조자료형 : 변수가 값을 가지고 있지않고 값이 있는곳을 참조
    
   *기본자료형(8가지)
   -정수를 표현하는 자료형(4가지) : byte, short, int, long
   -실수를 표현하는 자료형(2가지) : double,float
   -문자를 표현하는 자료형 : char(하나의 문자는 홋따옴표''로 묶어야함.)
   -논리를 표현하는 자료형 : boolean (true,false값)

      height = 166.8f;//실수값을 float변수에 저장하기 위해서는 접미사f
      height = (float)166.8; //(자료형)을쓰면 자료형 변환하여 저장할 수 있다

      10진수를
      -> 앞에 0을붙여 8진수로 표현
      -> 0x를붙여 16진수로 표현
      -> 0b를 붙여 2진수로 표현
      
      제곱승
      double data = 1.0E6;   //1.0 곱하기 10의 6승을 의미
      double data2 = 1.0E-6;  // 1.0 곱하기 10의 -6승을 의미


      자료형의 최대값을 넘으면 overFlow 됐다.
      자료형의 최소값보다 작으면 underFlow 됐다.

@연산자
1.산술연산자 2.증감연산자 3.비교연산자 4.논리연산자
5.비트연산자 6.복합치환연산자 7.삼항연산자

   *산술연산자 : +(더하기) -(빼기) *(곱하기) /(나누기) %(나머지값)
   *증감연산자 : ++ (변수자신에 더하기 1을하고 그 값을 다시 변수에 대입)
                      -- (변수자신에 빼기 1을하고 그 값을 다시 변수에 대입)
	       
	       증감연산자는 변수 앞뒤로 올수 있고 단독으로 쓸때는 결과가 같게 나옴

	       하지만 다른 연산자나 명령문에 사용이될때에는
	       변수명 앞에 오면 -> 먼저 변수를 증감하고 다른 연산자,명령문 수행
	       변수명 뒤에 오면 -> 다른 연산자,명령문을 실행하고 나중에 변수를 증감.

	       !) i+1=i+1
	          ++i => i = i+1 //i+1과 ++i는 다르다!
